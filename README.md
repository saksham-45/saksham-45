## ▶ &nbsp;[ FEATURED_WORK ]

> The six pinned repos below — skim the headline, dig into whichever pulls you.

### ◢ AI · Simulation · Signals

**[automix](https://github.com/saksham-45/automix-_.)** — Neural DJ mixing: a decision NN + LSTM curve generator

- Automated DJ engine that runs harmonic (key/tempo) analysis, micro-timing alignment, and stem-aware blending so transitions land without vocal clashes.
- A decision neural net picks the move; an LSTM predicts the transition curve, scored over spectral balance, loudness envelope, and rhythmic transients.
- `Python` · `PyTorch` · `Librosa` · `Demucs` · `Audio DSP`

**[introvertensemble](https://github.com/saksham-45/introvertensemble)** — Agent-based library-seating simulation

- Asks "in a library full of introverts, where does an agent sit?" — 8 student personas reacting to occupancy density, proximity, and re-seating pressure.
- Multi-factor seat scoring, graph-based routing, and 4 stochastic event types; seeded runs + telemetry make focal-agent behavior reproducible.
- `Python` · `Graph Algorithms` · `Simulation Heuristics`

**[MoodScope](https://github.com/saksham-45/MoodScope)** — Real-time emotion detection + LLM wellness insights

- Detects faces (MTCNN / SSD) and classifies emotion with a trained CNN over live webcam or video, logging to SQLite behind a PyQt5 dashboard with Matplotlib trends.
- Generates personalized wellness insights via a local Llama 3.2 (Ollama), with a rule-based fallback when the model isn't running.
- `Python` · `TensorFlow / Keras` · `OpenCV` · `PyQt5` · `Ollama`

### ◣ Systems · Compilers · Computational Math

**[Mathon](https://github.com/saksham-45/Mathon)** — LLVM-backed scientific-computing DSL & compiler

- Full pipeline — lexer → recursive-descent parser → AST → optimization → **LLVM IR (llvmlite)** → **JIT** — for a math DSL, with an `--explain` mode that prints every stage end to end.
- Symbolic differentiation on the AST, forward-mode autodiff via dual numbers, Simpson's-rule integration, and interactive plotting.
- `Python` · `LLVM / llvmlite` · `AST Lowering` · `matplotlib`

**[isomercount](https://github.com/saksham-45/isomercount)** — Combinatorial enumeration & research engine

- Exact counting for cyclic graphs and colored snake polyominoes using transfer matrices, Burnside's lemma, and Möbius-based filtering — never brute force.
- Automated recurrence discovery plus symbolic verification against known sequences.
- `Python` · `NumPy` · `SymPy` · `Combinatorics` · `LaTeX`

**[visualizzer](https://github.com/saksham-45/visualizzer)** — Real-time audio-reactive graphics engine &nbsp;·&nbsp; [live demo ↗](https://visualizzer.vercel.app/)

- Turns mic / system audio into reactive visuals: a 2048-point FFT drives 2D canvas, WebGL2 shaders, GPU particles, and Three.js 3D fluid scenes with intelligent visualizer switching.
- Math all the way down — Fourier analysis, linear-algebra projections, HSV↔RGB, complex-number shader rotations — held at 60fps.
- `JavaScript` · `Web Audio API` · `WebGL2` · `Three.js`
